# DevNest 

DevNest is a full-stack web application built with **ASP.NET**, **React**, and **SQL**, designed for learning and practicing modern web development concepts.

---

##  Tech Stack

### Backend
- **ASP.NET (.NET Web API)**
- **C#**
- **Entity Framework Core**
- **SQL Database**

### Frontend
- **React**
- **JavaScript**
- **HTML**
- **CSS**

---

##  Features
- RESTful API
- User authentication (Register / Login)
- DTO-based architecture
- Service & Repository pattern
- React frontend consuming ASP.NET API
- Clean separation between backend and frontend

---
